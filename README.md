# robot-lab - 机器人快速实验平台

利用各种开放软件，快速实现机器人原型的实验项目。

## 项目目标

- 快速验证机器人概念和设计
- 整合开源软硬件资源
- 降低机器人开发门槛
- 积累可复用的原型方案

## 目录结构

```
robot-lab/
├── README.md           # 项目说明
├── docs/               # 文档和笔记
│   ├── hardware/       # 硬件选型记录
│   ├── software/       # 软件工具链
│   └── experiments/    # 实验记录
├── prototypes/         # 原型项目
│   ├── arm/            # 机械臂
│   ├── mobile/         # 移动机器人
│   ├── gripper/        # 末端执行器
│   └── vision/         # 视觉系统
├── tools/              # 工具脚本
│   ├── setup/          # 环境配置
│   ├── simulation/     # 仿真工具
│   └── utils/          # 实用脚本
└── resources/          # 资源收集
    ├── cad/            # CAD模型
    ├── code/           # 代码片段
    └── refs/           # 参考资料
```
